    guictrl.button – Button Controls¶

    Button control implementations including standard buttons and toolbar-integrated buttons.

    button ¶

    Button widgets for the SchForm GUI framework.

    Provides button base classes, toolbar button variants, and bitmap/nobg/plate/menu button implementations.

    Modules¶

    base ¶

    Button widget classes for the SchForm GUI framework.

    Provides factory functions that create wxPython button subclasses integrated with the SchBaseCtrl mixin. All button classes are created dynamically through factory functions to reduce code duplication while allowing each button type to customize its behavior (bitmap handling, close behavior, menu handling, etc.).

    Factory functions

    _make_button_class, _make_menu_button_class
