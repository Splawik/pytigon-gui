    guilib.events – Event System¶

    Custom event types and event handling infrastructure for inter-component communication.

    events ¶

    Module containing all user-defined wx event IDs.

    Provides a centralized registry of wx.Window.NewControlId() identifiers used throughout the application for menu commands, toolbar actions, and custom events.
